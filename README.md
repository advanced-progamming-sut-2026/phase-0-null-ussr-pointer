## 🧟‍♂️ About The Project

Welcome to the **NULL USSR POINTER** repository. This project is a complete, ground-up recreation of the beloved *Plants vs. Zombies 2* mechanics, developed by **Group 4**.

We've focused on delivering robust gameplay mechanics wrapped in a clean, highly optimized Java architecture. By leveraging strict **MVC (Model-View-Controller)** principles and implementing core structural **Design Patterns**, the codebase is built to be scalable, readable, and highly resistant to those pesky memory leaks (and zombies).

## 🪴 Key Features & Mechanics

* **Grid-Based Combat System:** Accurate collision detection and lane management.
* **Sun Economy:** Dynamic resource generation and spending mechanics.
* **Horde AI:** Zombies with varying speeds, health pools, and unique abilities.
* **Clean Code:** Refactored for maximum efficiency, utilizing Factory, Strategy, and Builder patterns where appropriate.

## ⚙️ Initialization (Getting Started)

Since this repository is currently our public-facing mirror (syncing just a few days behind our main operational branch), you can easily clone and compile the defense systems yourself.

```bash
# 1. Clone the public mirror
git clone [https://github.com/advanced-progamming-sut-2026/phase-0-null-ussr-pointer.git](https://github.com/advanced-progamming-sut-2026/phase-0-null-ussr-pointer.git)

# 2. Navigate to the project directory
cd phase-0-null-ussr-pointer

# 3. Build the project using Gradle
./gradlew build

# 4. Deploy the defense systems (Run)
./gradlew run
